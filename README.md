## Descripción
Este proyecto consiste en un módulo CRUD desarrollado en C# Windows Forms para la gestión de usuarios dentro de un sistema de biblioteca.

El sistema permite:
- Crear usuarios
- Actualizar usuarios
- Eliminar usuarios
- Visualizar usuarios registrados
- Asignar roles o privilegios

## Roles
- Administrador
- Bibliotecario
- Estudiante

## Arquitectura utilizada
El proyecto implementa una arquitectura Cliente-Servidor.

- Cliente:
  La interfaz gráfica desarrollada en Windows Forms.

- Servidor:
  La lógica desarrollada en C# encargada de procesar las operaciones CRUD.

- Base de Datos:
  SQL Server utilizado para almacenar la información de los usuarios.

## Tecnologías utilizadas
- C#
- Windows Forms
- SQL Server
- Visual Studio
